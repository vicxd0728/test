# 壓力單位轉換器

這是一個簡單的前端工具，可將壓力數值在常見單位之間互相換算，並自動保留歷史詢問紀錄。

## 功能特色

- 支援帕、千帕、兆帕、巴、大氣壓、psi、托、毫米汞柱等多種壓力單位。
- 即時計算換算結果，並以醒目的文字顯示。
- 自動儲存最多 20 筆詢問紀錄，可隨時清除。
- 介面採響應式設計，於手機或桌機上皆可使用。

## 使用方式

1. 直接開啟 `index.html`，瀏覽器即會載入工具介面。
2. 輸入欲換算的壓力數值，選擇原始單位及目標單位後按下「開始轉換」。
3. 下方會顯示換算結果與歷史紀錄；如需清除，可按「清除紀錄」。

> 若瀏覽器允許使用 `localStorage`，歷史紀錄會在重新整理後保留；否則工具仍可正常換算，但不會保存紀錄。
