# 壓力單位轉換器

這是一個簡潔的前端介面，可在不同壓力單位之間快速轉換，並保留最近 20 筆的詢問紀錄。

## 使用方式

1. 直接以瀏覽器開啟 `index.html`。
2. 輸入壓力值並選擇原始與目標單位。
3. 按下「開始轉換」即可看到結果，所有查詢都會同步記錄於下方列表，並儲存在瀏覽器的本機儲存中。
4. 可隨時使用「清除紀錄」按鈕刪除現有紀錄。

## 支援單位

- 帕斯卡 (Pa)
- 千帕 (kPa)
- 百萬帕 (MPa)
- 巴 (bar)
- 標準大氣壓 (atm)
- 磅力每平方英吋 (psi)
- 毫米汞柱 (mmHg)

## 注意事項

- 當原始單位與目標單位相同時，系統會提醒重新選擇。
- 若瀏覽器停用本機儲存或隱私模式阻擋儲存，詢問紀錄將不會被保存。
